New Website/Online Catalog Launched
The county's library patrons may notice a new look to the the federation's website, http://www.ccfpl.org.  The site redesign corresponds with the unveiling of a brand new online catalog system (Horizon Information Portal) that vastly improves on the old catalog (known as Webpac).  In addition to the many new ways to search the libraries holdings, "HIP" also offers a more reliable design that will greatly improve availability to the new system from home.  The new search interface lets patrons limit searches to specific libraries and collections, as well as have increased access to account information and temporarily collect and save "booklists."  Look for more to be added to this powerful new system in the months to come.

 

Assistive Technology Grant Benefits Libraries & Patrons
Centre County library patrons with sight or hearing impairments, learning disabilities, or physical disabilities involving limited hand movement, will benefit from assistive technology recently installed in their local libraries.

Assistive technology is equipment and computer programs that allow people with special requirements to use a computer to run various programs, view the library catalog, and access the Internet.

Each library in the Centre County Federation of Public Libraries (CCFPL) - Aaronsburg Area Public Library, Centre County Library, Centre Hall Area Library, Holt Memorial Library, Schlow Centre Region Library and the Centre County Bookmobile - has installed assistive technology, which was funded by a federal Library Services and Technology (LSTA) grant.

The libraries and the bookmobile are equipped with screen magnifiers, screen readers, speech synthesizers, ergonomic mouse, and special keyboards, all of which will aid patrons with disabilities. Each computer workstation is set-up on a special table that will raise and lower for wheelchair compatibility. In addition, each library will soon have scanners that allow any type of printed material to be scanned into the computer and read to the user.
 

 

Centre County Library Added to Schlow Remote Book Returns
For your convenience, Schlow Centre Region Library’s remote book returns will now accept materials from Centre County Library, including all branches and the bookmobile.

Schlow Library has numerous remote book return locations throughout the Centre Region. Materials will be picked up every day, Monday through Saturday. One or two day loan items (reference books, popular videos) are to be returned directly to the library. The remote returns are available during store hours.
